Inugami-NG : inu-main-header

A menu bar containing the application's primary links is a standard feature in many applications. To meet this need, the inu-main-header component is available.

The showcase menu, which manages the various navigation links, is built using this component.

Usage

This component is designed to be defined directly in the main app.html file.

As with any standalone component, it must be imported directly into the component that uses it.

The InuSiteLink interface allows you to define the application's primary links.

title & path
The display name and the corresponding Angular route.
target
Supports standard HTML targets, including _blank for external windows.
children
Enables multi-level navigation or dropdown menus via the InuSiteLinkChildren interface.
gaEvent & gaCategory
Built-in support for Google Analytics tracking on link clicks.

The component is fully responsive. When the screen size becomes too small to display the full menu, a burger button appears, allowing users to toggle the menu.

Add actions

It is possible to add actions or content inside the component. These elements will be positioned on the right side of the component.

Inugami-NG : inu-page-layout

The inu-page-layout component is designed with a classic web page layout in mind, featuring a sidebar navigation on the left.

Inugami-NG was initially developed to meet the specific component and rendering needs of the Inugami website. Although this library has grown to be much more versatile, it was primarily built to simplify the design and development process of the Inugami platform as much as possible.

The showcase uses the inu-page-layout component within its main Angular template.

The component is fully responsive and adjusts its display mode for screens smaller than 920px. In this case, the sidebar menu is displayed above the main content.

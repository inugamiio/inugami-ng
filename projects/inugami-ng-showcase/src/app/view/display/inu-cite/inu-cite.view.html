Inugami-NG : inu-cite

A dedicated component for your documentation

The inu-cite component enables you to add citations to your applications by leveraging the semantic HTML5 cite tag. It is particularly useful for building documentation-based applications

Supported status levels

The inu-cite component supports several severity levels:

  • info
  • success
  • danger
  • warning

Inugami-NG : inu-toast

Show notifications within your apps

The inu-toast component allows you to display notifications in your applications as toast messages. These toasts are automatically dismissed after a specific delay.

To function correctly, the inu-toast component must be placed at the very end of your main Angular template.

To add notifications, you need to use the InuToastServices service.

The addMessage method takes a message parameter of type ToastMessage.

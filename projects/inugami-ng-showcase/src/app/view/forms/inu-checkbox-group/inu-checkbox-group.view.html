Inugami-NG : inu-checkbox-group

Simplified checkbox usage

The inu-checkbox-group component is designed to simplify checkbox usage within Signal-based forms. It leverages the InuSelectItem{{genericT()}} interface to provide a consistent and type-safe way to manage selection lists.

Example

Vertical example

Required example

Mandatory field logic is handled directly by the Signal form. Therefore, there is no need to define a required attribute on the component to enforce validation.

Disabled example

Similarly to mandatory fields, disabled states are also defined via the Signal form.

Inugami-NG : inu-svg-switzerland

The inu-svg-switzerland component provides an SVG map of Switzerland, allowing for the display and interaction of individual cantons

This component is also interactive, allowing users to select cantons and modify their associated values.

Example

In its simplest form, the component is limited to display only.

When hovering with the mouse, the outlines of the cantons are displayed. They can be selected with a click, which turns the outlines white. To deselect, hold the CTRL key and click.

Example monochrome

Customizing the color generation per canton is possible by defining the styleGenerator attribute.

Example handler

Displaying a map is a great start, but interactivity takes it to the next level.

The component is built for use within Signal Forms, a feature introduced in Angular 21.

    @for (colorName of colorNames(); track colorName; ) {
    • }

You can easily update the color palette by binding a signal to the styleGenerator attribute.

In this example, the actionHandler attribute truly comes into its own. We use a complex object to define the canton and its level. This object could vary significantly based on your business requirements. While the component gives you full control, it requires you to define both the actionHandler and the valueExtractor to ensure data is correctly extracted and color management remains functional.

The valueExtractor is straightforward, as it only needs to access the internal InuSelectItem object to retrieve the value.

Properties

Property Type Default value Description actionHandler InuSvgSwitzerlandAction

When the InuSelectItem contains a complex object or requires custom selection/deselection logic, a InuSvgSwitzerlandAction handler must be implemented.

disabled boolean false matcher InuSelectItemMatcher

A matcher must be used whenever the InuSelectItem contains a complex object to allow proper identification between the internal model and your model.

styleClass string inu-copy styleGenerator InuSvgSwitzerlandStyleGenerator

The styleGenerator attribute is used to provide a function that determines the style of each element according to its InuSelectItem value. It allows for dynamic canton coloring based on your business rules.

valueExtractor InuSvgSwitzerlandValueExtractor

In the case of complex objects, you must define an extractor to retrieve the value from the InuSelectItem. This value is then used by the styleGenerator to determine the appropriate colors.
